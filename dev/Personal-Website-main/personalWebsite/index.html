    Fast Lane

    FastLane is a self hosted serverless password manager software. Use only one password to get access to all of your passwords without storing all of them using a database. Uses a sha512 hashing algorithm and QuantomRandom to create and access passwords for any site using only your original master password that the program itself can also generate. This makes using a database to store your other passwords uncessesary and increases ease of use and security.
